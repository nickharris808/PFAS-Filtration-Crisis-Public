# Contributing to PFAS Filtration Crisis Documentation

Thank you for your interest in contributing to this public documentation project.

## Purpose of This Repository

This repository documents the PFAS contamination crisis using **publicly available information only**:
- EPA regulations and guidance documents
- Peer-reviewed scientific literature
- Public court filings and settlement documents
- Published manufacturer specifications
- Government contamination data

**No proprietary information is disclosed in this repository.**

## What We Accept

### Accepted Contributions

1. **Corrections**
   - Factual errors in cited data
   - Outdated regulatory information
   - Broken links or references

2. **Additional Public Data**
   - New EPA guidance documents
   - Published scientific studies
   - Public settlement announcements
   - State regulatory updates

3. **Formatting Improvements**
   - Typo corrections
   - Markdown formatting fixes
   - Accessibility improvements

4. **Translations**
   - Translations to other languages (with proper attribution)

### Not Accepted

1. **Proprietary Information**
   - Trade secrets
   - Unpublished research data
   - Confidential business information

2. **Promotional Content**
   - Product advertisements
   - Vendor comparisons
   - Marketing materials

3. **Speculative Content**
   - Unsubstantiated claims
   - Predictions without data
   - Personal opinions

## How to Contribute

### 1. Open an Issue

Before making changes, please open an issue describing:
- What you want to change
- Why the change is needed
- Source/citation for any new data

### 2. Fork and Branch

```bash
# Fork the repository on GitHub
# Clone your fork
git clone https://github.com/YOUR-USERNAME/PFAS-Filtration-Crisis.git

# Create a branch
git checkout -b fix/description-of-change
```

### 3. Make Changes

- Follow existing formatting conventions
- Add citations for all factual claims
- Update the "Last Updated" date if applicable

### 4. Submit Pull Request

- Reference the issue number
- Describe your changes clearly
- Include any relevant citations

## Citation Standards

All factual claims must include citations:

```markdown
According to EPA data, the MCL for PFOA is 4 ppt (EPA, 2024).

**Reference:**
EPA. "PFAS National Primary Drinking Water Regulation." 89 FR 32532 (April 26, 2024).
```

## Code of Conduct

- Be respectful and constructive
- Focus on facts, not opinions
- Respect intellectual property
- Maintain scientific objectivity

## Questions?

For questions about contributing, open an issue with the "question" label.

For questions about the proprietary technology referenced in this repository, contact the maintainers directly.
